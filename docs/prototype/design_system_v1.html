📋 Usage Guidelines

Color Usage
  • Use gradients for primary actions and hero elements
  • Maintain 4.5:1 contrast ratio for accessibility
  • Green indicates positive values/income
  • Red indicates negative values/expenses
  • Indigo for interactive elements and links
  • Gray scales for hierarchy and structure
Typography Guidelines
  • Use font weights 400, 600, and 700 only
  • Maintain clear hierarchy with size and weight
  • Keep line-height between 1.2-1.6
  • Use monospace fonts for monetary values
  • Ensure text is readable on dark backgrounds
Component Guidelines
  • Use consistent border-radius across components
  • Apply hover states to interactive elements
  • Maintain consistent spacing using the scale
  • Use gradients sparingly for emphasis
  • Ensure mobile-friendly touch targets (44px min)
Accessibility
  • Maintain WCAG 2.1 AA compliance
  • Use semantic HTML elements
  • Provide alt text for images and icons
  • Ensure keyboard navigation support
  • Test with screen readers
  • Use focus indicators for navigation
